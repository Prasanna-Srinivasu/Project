# "Tomato" - Food Delivery App UI

The frontend for "Tomato," a modern food ordering web application built entirely in React. This project showcases a dynamic and user-friendly interface that allows customers to easily browse menus and place orders.

### Key Responsibilities & Features

- **Dynamic Menu Display:** Created components to render menu items fetched from a live API.
- **Real-Time Cart Updates:** Implemented functionality for adding, removing, and updating item quantities with immediate UI feedback.
- **Client-Side Routing:** Used React Router to create a smooth, single-page application (SPA) experience.
- **Mobile-First Design:** Developed the layout with a mobile-first approach to guarantee a great experience on any device.

### Technologies Used

- **Frontend:** React, JavaScript, CSS3
- **Libraries:** React Router, Axios
